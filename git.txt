分支
    git clone:clone远程仓库
    git add . :添加所有改动的文件到暂存区
    git commit -m: 提交暂存区到本地仓库
    git push ：推送到远程仓库
    git branch：查看本地分支
    git branch -r：查看远程分支
    git branch -a：查看本地和远程的分支
    git checkout branch名：切换分支
    git branch 分支名：创建分支
    git branch -D 分支名：强制删除分支
    git merge 分支名：将分支合并到当前分支
